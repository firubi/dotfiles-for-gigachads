# Programs
- Distro - EndeavourOS with [CachyOS repos](https://wiki.cachyos.org/features/optimized_repos/)
- Shell - bash
- Terminal - kitty
- WM - Hyprland
- Launcher - tofi
- File manager - lf/dolphin
- Image viewer - oculante
- Video player - mpv
